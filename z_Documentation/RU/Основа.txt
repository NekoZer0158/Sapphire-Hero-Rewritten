Игра была создана в одиночку и выпущена 22 февраля 2024 года и поддерживалась обновлениями полтора года.
Из-за почти нулевой популярности игры никакого рефакторинга, чтобы сделать работу с игрой более приятной,
я не делал. Сама документация по той же причине не описывает вообще всё, а только основные моменты

Папки:
	Общее:
		addons - плагины (содержит только плагин для поддержки модов)
		Audio - музыка и звуки
		Fonts - шрифты
		Resources - ресурсы
		Scenes - сцены
		Scripts - скрипты
		Sprites - все 2D изображения (исключение: иконка игры)
		Translations - переводы
		z_Documentation - документация
	Audio:
		Music - музыка
		Sound - звуки
	Fonts - шрифт (он тут всего один)
	Resources:
		Animations - анимации (в папке представлены только сохранённые отдельно анимации, а не все)
		Databases - датабазы (проджектайлов). см. Проджектайлы.txt
		Modules - модули (в папке представлены только сохранённые отдельно модули, а не все)
		Other - всё остальное, что неподходит в другие папки, но и не заслуживает отдельных папок на уровне
		остальных. Содержит только Translation_loader.tres в папке Mods (никак не связан с пользовательскими
		модами). Подробнее про этот ресурс см. Перевод.txt.
		ShaderMaterials - папка для ShaderMaterial. Содержит только один ShaderMaterial - Customization_UI.tres
		Shaders - шейдеры
		Spriteframes - нужен для SpriteFrames, содержит только один файл - explosion_enemy.tres. (это не означает,
		что других SpriteFrames нету, просто это единственный сохранённый как отдельный файл)
		Themes - темы
		Tilesets - тайлсеты для уровней и сюжетных картинок (см. Тайлсеты.txt)
		Weapons - оружие игрока, врагов и боссов (см. Оружие)
	Scenes:
		Bodies - тела врагов и боссов
		Boss - боссы
		Effects - эффекты взрывов и партиклы
		Enemies - враги (см. Враги.txt)
		Filters - фильтры
		Levels - уровни:
			A1 - альтернативные уровни из первой (и единственной) альтернативной вселенной
		Minigames - мини-игры:
			Hack_planes - единственная мини-игра (shoot'em up)
		Objects - объекты (в том числе заднего плана):
			Background - объекты заднего плана (некоторые)
			Checkpoints - чекпоинты
			Clouds - облака
			Doors - двери
			Levers - рычаги
			Moving_platforms - движущиеся платформы
			Pickups - пикапы (содержит только один - хилку)
			Teleports - телепорты (содержит только один, изначально планировалось больше)
			Unique - уникальные (объекты только на один уровень, с которыми можно взаимодействовать)
		Other - всё то, что неподошло для других папок
		Player - игрок
		Projectiles - проджектайлы
		Story - всё, связанное с сюжетом
		UI - интерфейс
	Scripts:
		AI - ИИ врагов, боссов и для персонажа из катсцен
		Bodies - скрипты для тел персонажей
		Effects - эффекты взрывов и партиклы
		Levels - уровни:
			Loaders - загрузчики для пользовательских уровней
		Minigames - мини-игры (только одна)
		Objects - объекты
		Other - всё то, что неподошло для других папок
		Player - игрок
		Projectiles - проджектайлы
		Refcounted - скрипты, класса RefCounted
		Resources - скрипты для ресурсов:
			Bodies - ресуры для тел персонажей
			Databases - датабазы проджектайлов
		Singletons - синглтоны
		Story - скрипты для сюжета
		UI - интерфейс
	Sprites:
		Background - элементы заднего плана (облака, деревья и Солнце с Луной)
		Boss - боссы
		Effects - эффекты взрывов и огня летающих роботов
		Enemies - враги
		Icons - иконки для классов
		Minigames - мини-игры (только одна)
		Objects - объекты
		Other - всё то, что неподошло для других папок:
			Mobile - интерфейс для мобильной версии
			Unknown - неизвестый робот (не путать с боссом Неизвестным)
		Player - игрок
		Projectiles - проджектайлы
		Story - история
		Tilesets - тайлсеты
		UI - интерфейс:
			Achievements - достижения
			Black_Baron - Чёрный Барон (только текстура фаз)
			Controls - иконки для DualShock
			Main_menu - текстуры для главного меню
			Player - текстуры интерфейса игрока, включая скины
	Translations - перевод (см. Перевод.txt)
	z_Documentation:
		FAQ - ответы на вопросы
		Враги - информация о врагах
		Оружие - информация об оружии
		Основа - база, проясняющая основы
		Перевод - всё о переводе
		Проджектайлы - всё о проджектайлах
		Тайлсеты - всё о тайлсетах

Синглтоны:
	GlobalSapphire - главный синглтон, содержащий необходимую информацию о проджектайлах, игроке, музыке и т.д.
	GlobalEnum - содержит только Enum

Персонажи:
	ИИ (мозг) и тело отделены, позволяя дать любое тело игроку, либо врагу.
	Сцены тел находятся в папке Scenes/Bodies, кроме тела игрока, оно находится в папке Player
