Типы врагов:
	1) Static - статичные (турели)
	2) Moving - движущиеся (солдаты, мины и катающиеся роботы)
	3) Flying - летающие

Классы:
	Есть два основных класса для тел врагов:
		1) Robot_body - используется не только для тел врагов (наследует класс CharacterBody2D)
		2) Static_Robot_body - только для тел турелей (наследует класс StaticBody2D)
	Для тел летающих врагов берётся за основу класс Flying_body (наследует класс Robody_body)

Состояния врагов солдат:
	IDLE - ходит туда сюда на небольшой территории
	ATTACKING - атакует игрока
	WORRIED - начинать бродить по более обширной территории, чем прежде, ведь знает, что игрок рядом

Исключения:
	Ice_spider - из тела сразу делается враг
	Unknown_cutscene и Unknown_cutscene_2 - используются только в катсценах, сразиться с ними нельзя

Пути:

	Скрипты тел: Scripts/Bodies
	Текстуры врагов: Sprites/Enemies
